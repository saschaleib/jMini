jMini Download Builder

This page helps you to customize jMini to your needs: simply select the functions that you need for your project, and click on Download to get a tailor-made version with only the required functionality.

Note: This page is stil work in progress and is not fully functional yet.

This page is powered by jMini ... though at the present time, there are still some functions that are missing, and they have to be imported from jQuery instead. The long-term plan is to replace them with own functions.

For more information on jMini, please see the GitHub Repository.
