<Number>.toBytesString(…)

Formats a number as Bytes multiplies string.

Usage:

<String> = <Number>.toBytesString(digits <Number> = 2, locale <String>|<Intl.Locale> = 'en-US')
Member of:
<Number>, any valid Number.
Parameters:
  1. digits – the number of significant digits, as used by the Intl.NumberFormat function (optional, default: 2).
  2. locale – a valid BCP 47 language tag, or an Intl.Locale instance (optional, default: 'en-US').
Returns:
A <String> representing the number of bytes, shown in an appropriate unit and locale.

Notes:

  1. This function uses base-1024 IEC units (KiB, MiB) for output. The maximum unit magnitude supported is YiB (Yobibyte)
  2. The switch to the next higher unit happens at around 0.90 of the higher unit (see examples below).
  3. This function places a narrow no-break space (U+202F) between the number and the unit.
  4. JavaScript interprets a period in numerals as decimal point. To directly call this function on number literals, please enclose them in brackets (see below).

Examples:

Example with default values:

console.log (1010).toBytesString();  // Output: "0.99 KiB"

Examples with options:

(1020001).toBytesString(7);  // Result: "0.9727488 MiB" (1010).toBytesString(2, 'de');  // Result: "0,99 KiB" (256543000000).toBytesString(2,'ar');  // Result: "٢٤٠ MiB"
