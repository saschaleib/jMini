document.onReady(…)

Attaches an event handler callback function to the “DOMContentLoaded” event of the document.

Usage:

<document> = document.onReady(callback <Function>)
Member of:
The document element (static method).
Parameters:
  1. callback – the function to call when the event occurs (<Function>, required).
Returns:
The document element, thus allowing for command chaining.
