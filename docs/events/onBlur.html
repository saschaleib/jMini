<HTMLElement>.onBlur(…)

Attaches an event handler callback function to the “blur” event of the HTMLElement.

Usage:

<HTMLElement> = <HTMLElement>.onBlur(callback <Function>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. callback – the function to call when the “blur” event occurs (<Function>, required).
Returns:
The same <HTMLElement> that it was called on, thus allowing for command chaining.

Notes

  1. This is just a shorthand function for the .on method, specific for the “blur” event. See there for more information and examples.
  2. The “blur” event handler can be attached to any HTMLElement, but of course only makes sense for elements which are either naturally focusable (e.g. <a>, <input>, <button>, <textarea>, etc.), or are otherwise made focusable (e.g. by setting the tabindex CSS property). Also note that focus may behaviour may not be the same across different platforms.
