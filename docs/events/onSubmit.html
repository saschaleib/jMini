<HTMLFormElement>.onSubmit(…)

Attaches an event handler callback function to the “submit” event of a <form> element.

Usage:

<HTMLFormElement> = <HTMLFormElement>.onSubmit(callback <Function>)
Member of:
<HTMLFormElement>, i.e. the <form> element.
Parameters:
  1. callback – the function to call when the “submit” event occurs (<Function>, required).
Returns:
The same <HTMLFormElement> that it was called on, thus allowing for command chaining.

Notes

  1. This is just a shorthand function for the .on method, specific for the “submit” event. See there for more information and examples.
  2. The onSubmit method can only be called on instances of the <form> element.
