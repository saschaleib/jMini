<HTMLElement>.once(…)

Attaches a callback function for a named event to an HTMLElement. The callback will be automatically removed after it has been called.

Usage:

<HTMLElement> = <HTMLElement>.once(name <String>, callback <Function>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. name – the name of the event to attach to (<String>, required).
    Examples: 'click', 'hover', etc.
  2. callback – the function to call when the event occurs (<Function>, required).
Returns:
The same <HTMLElement> that it was called on, thus allowing for command chaining.

Notes

  1. This is functionally identical to the .on method and uses similar code. See there for more information and more examples.

Examples

let foo document.getElementById('foo'); foo.once('click', function(e) { console.log(e); });
