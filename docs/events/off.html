<HTMLElement>.off(…)

Removes a previously attached event callback function from an HTMLElement.

Usage:

<HTMLElement> = <HTMLElement>.on(name <String>, callback <Function>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. name – the name of the event to remove the handler from (<String>, required).
  2. callback – a reference to the function that should be removed (<Function>, required).
Returns:
The same <HTMLElement> that it was called on, thus allowing for command chaining.

Notes

  1. This is just a simple wrapper for the JavaScript .removeEventListener method, which also allows to use more specific options.
  2. The function reference needs to be the same as was previously attached using the .on, or any of the shortcut functions. This implies that such a reference must be available, which means that anonymous functions can not be detached.

Example

let callback = function(e) { console.log(e); } let foo document.getElementById('foo'); foo.on('click', callback);   // attaches the callback function … foo.off('click', callback); // removes the function again.
