<HTMLElement>.onClick(…)

Attaches an event handler callback function to the “click” event of the HTMLElement.

Usage:

<HTMLElement> = <HTMLElement>.onClick(callback <Function>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. callback – the function to call when the Click event occurs (<Function>, required).
Returns:
The same <HTMLElement> that it was called on, thus allowing for command chaining.

Notes

  1. This is just a shorthand function for the .on method, specific for the “click” event. See there for more information and examples.
