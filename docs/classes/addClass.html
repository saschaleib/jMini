.addClass

Attaches a class to an HTMLElement.

Usage:

<HTMLElement>.addClass(className <String>)

Where:

  • <HTMLElement> is any valid HTMLElement
  • className <String> is the name of the class to add as String object, or quoted string

Note: This function is a simple wrapper for the classList.add() function. The only difference is that this function returns the HTMLElement and thus allows chaining other functions for neater and/or more compact code.

Examples

Simple example:

let trg = document.getElementById('target'); trg.addClass('redOutline');

Chaining example:

Chaining allows for very compact code:

document.getElementById('target').addClass('redOutline').attr('data-value', 'something');

… but also allows for very neat and readable code:

document.getElementById('target') .addClass('redOutline') .attr('data-value', 'something') ;
