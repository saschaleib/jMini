Classes (Topic)

This topic contains all functions that are directly related to Classes:

Add/remove

  • <HTMLElement>.addClass() – attach a class.
  • <HTMLElement>.removeClass() – remove a class.
  • <HTMLElement>.toggleClass() – toggles (attaches or removes) a class.

Checking

  • <HTMLElement>.hasClass() – checks if an a class is assigned.
