Classes (Topic)

This topic contains all functions that are directly related to Classes:

  • .addClass() – attach a class to an HTMLElement.
  • .hasClass() – Checks if an HTMLElement has a specific class assigned.
  • .removeClass() – removes a class from an HTMLElement.
  • .toggleClass() – toggles (attaches or removes) a class of an HTMLElement.
