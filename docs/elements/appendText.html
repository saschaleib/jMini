.appendText

Appends a new Text node to the parent element (as the last child element).

Usage:

<HTMLElement> = <HTMLElement>.appendText(txt <String>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. txt – the text to add (<String>, required).
Returns:
The parent <HTMLElement>, allowing for command chaining.

Note: This function encodes HTML special characters (<, >, " and ') before creating the text node, thus HTML code passed as parameter will be displayed as text, not parsed into a DOM structure. If you want to parse HTML code, please use the .setHtml function instead.

Examples

Simple example:

let foo = document.getElementById('foo'); foo.appendText('Hello world!');
