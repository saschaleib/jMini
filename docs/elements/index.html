Elements (Topic)

This topic contains all functions that are directly related to HTMLElements:

Attributes

  • <HTMLElement>.getAttr() – returns the current value of an attribute of the HTMLElement.
  • <HTMLElement>.setAttr() – sets the value of an attribute of the HTMLElement.

Content

  • <HTMLElement>.appendText() – appends a new text node as the last child of the HTMLElement.
  • <HTMLElement>.prependText() – prepends a new text node as the first child of the HTMLElement.
  • <HTMLElement>.getText() – returns the entire text content of the HTMLElement (including from child nodes).
  • <HTMLElement>.setText() – sets the entire tet value of the HTMLElement (overriding existing child nodes).

Insertion

  • <HTMLElement>.append() – attach an HTMLElement as child element, after any already existing child elements.
  • <HTMLElement>.prepend() – attach an HTMLElement as child element, before any already existing child elements.

New elements

  • HTMLElement.new() – creates a new HTMLElement [static method].
  • <HTMLElement>.appendNew() – creates a new HTMLElement and attaches as a child element (at the end)
  • <HTMLElement>.prependNew() – creates a new HTMLElement and attaches as a child element (at the begining)
  • <HTMLElement>.html() – parses XHTML and inserts the result as innerHTML.
