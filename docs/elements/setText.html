.setText

Sets the text content of the HTMLElement it is called on, possibly overriding any existing text or element nodes.

Usage:

<HTMLElement> = <HTMLElement>.setHtml(text <String>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. text – the text content to set (<String>, required).
Returns:
The same <HTMLElement> that it was called on, thus allowing for command chaining.

Note: This function encodes HTML special characters (<, >, " and ') before creating the text node, thus HTML code passed as parameter will be displayed as text, not parsed into a DOM structure. If you want to parse HTML code, please use the .setHtml function instead.

Examples

Simple example:

let foo = document.getElementById('foo'); foo.setText('Hello, world!');
