<HTMLElement>.load()

Loads an HTML snippet into an element.

Usage:

<HTMLElement> = <HTMLElement>.load(url, options)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. url – the address to load the HTML snippet from.
  2. options – an object containing options for the HTTP request, as specified for the fetch() function.
Returns:
The parent element.

Notes:

  1. This method returns the element that it was called on, thus allowing for command chaining.

Examples:

A simple example could look like this:

document.getElementById('foo') .load("bar.html") .show() ;
