.hasClass

Checks if an HTMLElement has a specific class atached.

Usage:

<Boolean> = <HTMLElement>.hasClass(className <String>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. className – the name of the class to check for, as String object or as quoted string (required).
Returns:
<Boolean> – true if the class is attached, false if it is not.

Note: This function is a simple wrapper for the classList.contains() function and only provides a slighly more convenient way of referencing this property.

Examples

Simple example:

let foo = document.getElementById('foo'); if (foo.hasClass('bar')) {  … };
