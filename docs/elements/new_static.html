HTMLElement.new (static)

Creates a new <HTMLElement> of the given name.

Usage:

<HTMLElement> = HTMLElement.new(name <String>, attrlist <Object>)
Member of:
HTMLElement (static method).
Parameters:
  1. name – the name of the element to create (<Strig>, required).
  2. attrlist – a flat Object list, containing names and values of attributes to add (optional).
Returns:
The new <HTMLElement> for chaining.

Examples

Simple example:

let foo = HTMLElement.new('div');

Example with attributes:

let bar = HTMLElement.new('a', { 'href': 'http://code.kolmio.com/jmini/', 'class': 'bar', 'target': '_blank', 'hreflang': 'en' });
