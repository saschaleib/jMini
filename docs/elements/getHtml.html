<HTMLElement>.getHtml()

Returns the inner HTML of the <HTMLElement> (including sub-elements) as <String>.

Usage:

<String> = <HTMLElement>.getHtml()
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
none
Returns:
The inner HTML of the parent <HTMLElement> as <String>.

Examples

Simple example:

console.log document.getElementById('foo').getHtml();
