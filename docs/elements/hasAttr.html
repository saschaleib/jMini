<HTMLElement>.hasAttr(…)

Checks if an HTMLElement has a specific attribute.

Usage:

<Boolean> = <HTMLElement>.hasAttr(attrName <String>, attrValue <String>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. attrName – the name of the attribute to check for (<String>, required).
Returns:
<Boolean> – true if the attribute exists.

Notes:

  1. This function can be used to check if empty attributes (like hidden, open, etc.) are present, which is not possible with the .getAttr function.
  2. This is just a simple wrapper for the JavaScript hasAttribute method. If you only need this function relatively few times, it is more efficient to use .hasAttribute(…) instead.

Example

let foo = document.getElementById('foo'); if (foo.hasAttr('bar')) { … }
