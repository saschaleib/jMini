<HTMLElement>.appendNew(…)

Appends a new <HTMLElement> to the parent element.

Usage:

<HTMLElement> = <HTMLElement>.appendNew(name <String>, attrlist <Object>)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. name – the name of the HTMLElement to create (required).
  2. attrlist – a flat Object list, containing names and values of attributes to add (optional).
Returns:
The newly created <HTMLElement> that it was called on, thus allowing for command chaining.

Important: Different to many other functions, this one returns the new HTMLElement that was created, not the one it was called on!

Note: This function uses the HTMLElement.new function, which also needs to be included.

Examples

Simple example:

let foo = document.getElementById('foo'); foo.appendNew('div');

Example with attributes:

document.getElementById('foo') .appendNew('a', { 'href': 'http://code.kolmio.com/jmini/', 'class': 'bar', 'target': '_blank', 'hreflang': 'en' } );
