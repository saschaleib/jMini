$p (Page Framework)

This is a simple page-based framework for web pages scripts. It is very much a “work in progress” and contains only very limited functionality:

Core

  • $p – The framework core (required for all other modules).

GUI

  • $p.gui – The core for all GUI functions.
  • $p.gui.tabs – simple handling for tab-based interfaces.
