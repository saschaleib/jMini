$p.url (URL Core)

This is the core module for URL funtionalities in page framework. It is required for all the URL modules in the $p page framework.

The only purpose of this module is to pass the core module’s pre-init functionality on to the sub-modules. I.e. during the pre-init phase, it will call any _init method of all sub-modules that it can find.
