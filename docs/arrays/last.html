<Array>.last()

Returns the array’s last element, or null if the array was empty.

Usage:

<any> = <Array>.last()
Member of:
<Array>, an Array of any type.
Returns:
The last element that was found in the array, without changing it’s type.

Examples:

A simple example could look like this:

let foo = document.getElementsByTagName('foo'); let bar = foo.last(); if (bar) { … }
