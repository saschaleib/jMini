<HTMLElement>.getSiblings(…)

Get a list of the Element‘s siblings, optionally filtered by a callback function.

Usage:

<Array> = <HTMLElement>.getSiblings(callback <Function> = undefined)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. callback – a function that returns true, if an item should be added (<Function>, optional).
Returns:
An <Array> of <HTMLElement>s.

Notes:

  1. The resulting list will exclude the item that it was called on itself.

Examples:

A simple example could look like this:

document.getElementById('foo') .getSiblings() .forEach( (item) => { … });

By using a filtering function, e.g. to only collect siblings which are <a> elements:

let list = document.getElementById('foo') .getSiblings( (element) => { return (element.nodeName == 'A'); });

Note that the .nodeName property always contains the element name in uppercase letters!
