<HTMLElement>.getChildren(…)

Get a list of the Element’s immediate children, optionally filtered by a callback function.

Usage:

<Array> = <HTMLElement>.getChildren(callback <Function> = undefined)
Member of:
<HTMLElement>, any valid HTMLElement.
Parameters:
  1. callback – a function that returns true, if an item should be added (<Function>, optional).
Returns:
An <Array> of <HTMLElement>s.

Examples:

A simple example could look like this:

document.getElementById('foo') .getChildren() .forEach( (item) => { … });

By using a filtering function, e.g. to only collect dscendants which are <a> elements:

let list = document.getElementById('foo') .getChildren( (element) => { return (element.nodeName == 'A'); });

Note that the .nodeName property always contains the element name in uppercase letters!
